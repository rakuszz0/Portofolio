# Rahmat Ilahi - Digital Architect Portfolio

Website portofolio personal yang dibangun dengan tema **Futuristik & Cyber-Tech**, dirancang khusus untuk merefleksikan identitas sebagai **Backend-Heavy Fullstack Engineer**.

## 🚀 Tech Stack

### Frontend & UI/UX
- **React.js (Vite)**: Framework utama untuk performa tinggi.
- **Tailwind CSS**: Styling modern dengan kustomisasi utilitas tinggi.
- **Framer Motion**: Animasi tingkat lanjut dan micro-interactions.
- **Lucide React**: Set ikon vektor yang konsisten dan ringan.
- **Custom Cursor Logic**: Animasi kursor dinamis yang merespon elemen interaktif.

### Backend & Architecture (Core Competencies)
- **Go (Golang)**: Untuk sistem berperforma tinggi dan modular.
- **Node.js & Express.JS**: RESTful API development.
- **gRPC**: Komunikasi antar layanan yang efisien.
- **Databases**: MongoDB, Redis, PostgreSQL, MySQL.

### DevOps & Cloud
- **Docker & Kubernetes**: Kontainerisasi dan orkestrasi aplikasi.
- **AWS**: Infrastruktur cloud (EC2, ECS, EKS).

## 🛠️ Fitur Utama

- **Digital Command Center UI**: Desain asimetris yang unik dengan nuansa futuristik.
- **Interactive Terminal Access**: Emulator terminal fungsional di dalam website untuk akses data cepat via CLI.
- **Bento Grid Portfolio**: Layout portofolio yang dinamis dan terorganisir.
- **Glassmorphism & Neon Visuals**: Efek visual premium dengan tekstur grain/noise.
- **Smooth Navigation**: Anchor scroll otomatis dan redirect interaktif dari terminal.

## 📁 Struktur Proyek

- `/src/components`: Komponen modular (Navbar, Hero, About, Projects, TerminalModal, dll).
- `/src/lib`: Utility fungsi seperti `cn` untuk manajemen class Tailwind.
- `/src/index.css`: Pusat konfigurasi tema, filter visual, dan animasi kustom.
- `/public`: Aset publik (CV, favicon, dll).

## 💻 Cara Menjalankan

1. Clone repositori ini.
2. Jalankan `npm install` untuk memasang semua dependensi.
3. Jalankan `npm run dev` untuk memulai server pengembangan.
4. Buka `http://localhost:5173` di browser Anda.
